SubnetForge a subnet Calculator (FLSM and VLSM)
CpE students waste time manually computing subnets and then typing Cisco IOS commands by hand. One wrong value breaks the entire lab configuration.
This tool computes both Fixed-Length and Variable-Length Subnet Masks, displays a structured routing table, and auto-generates the exact CLI commands ready to paste into Packet Tracer.
Features

FLSM and VLSM subnet computation
Structured routing table output
Auto-generated Cisco IOS CLI commands for direct Packet Tracer use(supports cisco packet tracer ver5.3.3 below)
   - it can be used on the newest version of packet tracer if you are only using fast ethernet.
Fully offline, runs in any browser

How to run

Download Subnet-Calculator_v1.html
Open it in any browser
No installation needed

Tech used
HTML, CSS, JavaScript
